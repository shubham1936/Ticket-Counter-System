# Ticket Counter System

## Overview

This project is a simple Ticket Counter System built using Python. It demonstrates the implementation of the Queue data structure following the First In, First Out (FIFO) principle. The project allows customers to join a queue, be served in order, and view the current state of the queue.

This project was created as part of my Data Structures and Algorithms (DSA) learning journey.

## Features

- Add customers to the queue
- Serve customers in FIFO order
- Display the current queue
- Simple and easy-to-understand implementation
- Implemented using Python

## Data Structure Used

- Queue (FIFO)
